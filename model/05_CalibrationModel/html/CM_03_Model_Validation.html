# SPECIFIKACE TECHNOLOGICKHO PROCESU ZDN

brick_thickness = 440     # mm, tlouka zdicho prvku (Porotherm 440 Profi)
brick_height = 250        # mm, vka zdicho prvku (Porotherm 440 Profi)
brick_width = 250         # mm, ka zdicho prvku (Porotherm 440 Profi)

# SOUADNICE REFERENNHO BODU (nad verifikanm stolem)

refer_x = 2_000           # mm, souadnice X referennho bodu
refer_y = 3_500           # mm, souadnice Y referennho bodu
refer_z = 1_000           # mm, souadnice Z referennho bodu
In[9]:
def calculation_dist(x, y, z):
    """
    Funkce pro vpoet drhy trajektorie od referennho bodu k clov poloze prvku

    Parametry:
    x, y, z (int): souadnice clov polohy prvku [mm]

    Nvratov hodnota:
    dist (int): drha trajektorie od referennho bodu k clov poloze prvku [mm]
    """
    dist = 0

    # Fze 1: drha od referennho bodu k clov stn.
    dist = ((refer_z - (z + brick_height*2))**2 + (refer_x - brick_thickness//2)**2)**(1/2)

    # Fze 2: drha ve smru osy X
    if x != brick_thickness / 2:
        dist = dist + abs(x - brick_thickness//2 + brick_width * 2)

    # Fze 3: drha ve smru osy Y
    if y != brick_thickness / 2:
        dist = dist + abs(y - refer_y + brick_width * 2)

    return int(dist)

Celkov vyhodnocen shody simulovanch a relnch dat

Celkov validan analza prokzala vysokou mru shody mezi simulovanmi a relnmi daty ve vech posuzovanch charakteristikch. Bootstrapovan KolmogorovSmirnovv test indikoval, e v 73,1 % ppad nelze statisticky rozliit simulaci od reality, co potvrzuje dobrou distribun konzistenci modelu. Reln smrodatn odchylka (28,46) se nachz uvnit 95% intervalu spolehlivosti simulace [7,64; 30,93], piem model vykazuje mrn zvenou variabilitu, kter vak zstv statisticky pijateln. Reln prmr (41,80) le uvnit 95% intervalu simulace [38,12; 45,90]. Welchv t-test neprokzal statisticky vznamn rozdl mezi prmry (p = 0,905), Cohenovo d = 0,013 indikuje prakticky nulov efekt a neparametrick MannWhitneyv test rovn nepotvrdil rozdl mezi rozdlenmi (p = 0,186). Souhrnn lze konstatovat, e model reprodukuje reln data bez statisticky ani vcn vznamnch odchylek a je v analyzovanm rozsahu empiricky validovn.
