# Bootstrap vzorkovn z kalibrace na velikost relnho datasetu

Cohens d (velikost efektu)

In[20]:
mean_diff = abs(real_validation_df['total_time'].mean() - df_syn['total_time'].mean())

pooled_std = np.sqrt(
    (real_validation_df['total_time'].std()**2 + df_syn['total_time'].std()**2) / 2
)

d = mean_diff / pooled_std

print("Cohen's d:", d)

Cohen's d: 0.04051924047307539

Cohen's d < 0.2 zanedbateln vznam

p = 0.0072

Interpretace:

p < 0,05 lze zamtnout nulovou hypotzu.

Hodnota p-value = 0.0072 je ni ne bn uvaovan hladina vznamnosti ( = 0.05), a proto lze zamtnout nulovou hypotzu o shod rozdlen porovnvanch skupin. Vsledek naznauje statisticky vznamn rozdl mezi skupinami, kter me souviset s odlinm tvarem rozdlen nebo medinem sledovan veliiny.

Celkov zvr

Na zklad provedench statistickch test lze konstatovat, e simulovan data vykazuj dobrou shodu s relnmi daty z hlediska stedn hodnoty, co potvrzuje jak Welchv t-test (p = 0.7126), tak zanedbateln velikost efektu (Cohens d = 0.0405). Reln prmr se navc nachz v 95% intervalu spolehlivosti simulace. Bootstrap KS test dle ukzal, e ve 72.2 % ppad nelze simulovan a reln data statisticky rozliit. Reln smrodatn odchylka sice spad do intervalu simulace, simulovan model vak vykazuje tendenci k vy variabilit. Naproti tomu vsledek MannWhitneyova U testu (p = 0.0072) indikuje statisticky vznamn rozdl v rozdlen dat, pravdpodobn v dsledku odlinho tvaru rozdlen nebo medinu. Celkov tedy simulace dobe reprodukuje zkladn charakteristiky relnch dat, avak nemus pln zachycovat jejich distribun vlastnosti.
